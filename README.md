END TO END BANKING RISK AND KPI ANALYTICS

This project analyzes a $435M lending portfolio to derive actionable insights for credit risk assessment and business decision-making, using SQL and Python to evaluate loan performance, identify default patterns, and optimize credit strategy. A default risk of 13.8% was identified along with key correlations between borrower attributes such as Debt-to-Income ratio and repayment behavior, where strong relationships were observed with repayment rates reaching up to 86.2% for certain segments. Automated ETL pipelines were built to process 38K+ records, enabling efficient data cleaning, transformation, and reducing manual reporting effort, while ensuring consistency and scalability. Additionally, interactive Power BI dashboards were developed to monitor key KPIs including loan performance, customer segmentation, and seasonal trends, providing clear visibility into high-risk segments and supporting data-driven business decisions in financial risk management.
